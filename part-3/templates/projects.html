My Projects

This page shows how to loop through a list of dictionaries - a common pattern when displaying database data!

{% for project in projects %}
{{ project.name }}
Tech: {{ project.tech }}
{% if project.status == 'Completed' %} {{ project.status }} {% elif project.status == 'In Progress' %} {{ project.status }} {% else %} {{ project.status }} {% endif %}
{% endfor %}

The Data Structure

In app.py, we pass a list of dictionaries:

project_list = [
    {'name': 'Personal Website', 'status': 'Completed', 'tech': 'HTML/CSS'},
    {'name': 'Flask Blog', 'status': 'In Progress', 'tech': 'Python/Flask'},
    {'name': 'Weather App', 'status': 'Planned', 'tech': 'JavaScript'},
]

Accessing Dictionary Values

In Jinja2, you can access dictionary values with dot notation:

{{ project.name }}
{{ project.status }}
{{ project.tech }}

As a Table

{% for project in projects %} {% endfor %}
# Project Name Technology Status
