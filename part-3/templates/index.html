Welcome to Jinja2 Variables!

Hello, {{ name }}!
What just happened?

In app.py, we wrote:

return render_template('index.html', name=student_name)

And in this HTML, we display it with:

{% raw %}{{ name }}{% endraw %}

The Double Curly Braces {% raw %}{{ }}{% endraw %}

In Jinja2 templates, {% raw %}{{ }}{% endraw %} means "output this Python value here".

  • {% raw %}{{ name }}{% endraw %} - Outputs the value of name
  • {% raw %}{{ name|upper }}{% endraw %} - Outputs in UPPERCASE: {{ name|upper }}
  • {% raw %}{{ name|lower }}{% endraw %} - Outputs in lowercase: {{ name|lower }}
  • {% raw %}{{ name|length }}{% endraw %} - Length of the name: {{ name|length }} characters
