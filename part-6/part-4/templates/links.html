URL Building with url_for()

Instead of hardcoding URLs, Flask provides url_for() to generate URLs dynamically!

Why use url_for()?
  • URLs automatically update if you change route paths
  • Handles URL encoding for special characters
  • Makes your code more maintainable
  • Works in both Python and Jinja2 templates

Generated URLs

These URLs were generated in Python using url_for():

Using url_for() in Python

from flask import url_for

# Basic usage
url_for('home') # Returns '/'

# With parameters
url_for('user_profile', username='Alice') # Returns '/user/Alice'
url_for('show_post', post_id=42) # Returns '/post/42'

Using url_for() in Templates

<!-- Link to home -->
<a href="{{ url_for('home') }}">Home</a>

<!-- Link with parameter -->
<a href="{{ url_for('user_profile', username='Alice') }}">Alice</a>
