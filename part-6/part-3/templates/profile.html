User Profile

This page demonstrates passing multiple variables to a template.

{{ name }}

Age: {{ age }} years old
Course: {{ course }}
Status: {% if is_enrolled %} Enrolled {% else %} Not Enrolled {% endif %}

How It Works

In app.py, we pass multiple variables:

return render_template('profile.html',
    name='Sarah',
    age=22,
    course='Web Development',
    is_enrolled=True)

Jinja2 Conditionals

Notice the enrollment status badge? It uses a Jinja2 if-statement:

{% if is_enrolled %}
    <span class="badge-success">Enrolled</span>
{% else %}
    <span class="badge-warning">Not Enrolled</span>
{% endif %}
